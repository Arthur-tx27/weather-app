# Weather App

Приложение для просмотра прогноза погоды. Получает данные через [Open-Meteo API](https://open-meteo.com/).

## Логика взаимодействия

1. **Поиск города** — введите название в поле поиска. При вводе отображаются подсказки (до 5 вариантов), доступна клавиатурная навигация (↑↓ Enter Esc).
2. **Текущая погода** — после выбора города отображается карточка с температурой, иконкой погоды, названием города и датой.
3. **Дополнительные метрики** — в блоке под температурой: ощущаемая температура, влажность (%), скорость ветра, количество осадков.
4. **Прогноз на 7 дней** — дневной прогноз с максимальной/минимальной температурой и иконкой погоды для каждого дня.
5. **Почасовой прогноз** — прокручиваемый список с температурами по часам. Селектор дней позволяет переключаться между датами.
6. **Переключение единиц** — выпадающий список в шапке: метрические (°C, км/ч, мм) или имперские (°F, миль/ч, мм). Все значения пересчитываются автоматически.
7. **Адаптивность** — интерфейс подстраивается под мобильные (до 480px) и десктопные (от 768px) экраны.
8. **Интерактивные состояния** — все кликабельные элементы имеют стили при наведении (`:hover`) и фокусе (`:focus-visible`).

## Запуск

### Бэкенд

```bash
cd backend
npm install
npm run dev
```

Сервер запускается на `http://localhost:3001`.

### Фронтенд

```bash
cd frontend
npm install
npm run dev
```

Открыть `http://localhost:5173` в браузере.

## Стек

| Слой | Технологии |
|------|-----------|
| Бэкенд | Node.js, Express, TypeScript |
| Фронтенд | React 19, TypeScript, Vite |
| API | Open-Meteo (forecast + geocoding) |
| Тесты | Jest, ts-jest, Testing Library |
| Линтинг | ESLint, Prettier |

## Архитектура

- **Бэкенд** — модульный монолит: `modules/weather/` (controller, service, types, test), `shared/`
- **Фронтенд** — Feature-Sliced Design: `app/` → `pages/` → `widgets/` → `features/` → `entities/` → `shared/`
